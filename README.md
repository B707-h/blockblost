# Block Blost

A pygame-based web game deployed with GitHub Pages.

## Play Online

The game is deployed and can be played online at:

**https://b707-h.github.io/blockblost/**

## About

This is a pygame web application built using pygbag, which allows Python pygame games to run in the browser using WebAssembly.

## Files

- `index.html` - Main HTML file with embedded Python code
- `block_blost.apk` / `block_blost.tar.gz` - Game assets archive
- `favicon.png` - Website favicon

## Deployment

The site is automatically deployed to GitHub Pages using GitHub Actions whenever changes are pushed to the main branch.
